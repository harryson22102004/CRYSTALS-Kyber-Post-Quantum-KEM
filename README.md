# CRYSTALS-Kyber Post-Quantum KEM
A research-oriented simulation of the NIST-standardized Post-Quantum Cryptography algorithm.

### 🔬 Research Highlights
- [cite_start]**Ring-LWE Operations**: Implements the Module Learning-with-Errors (MLWE) problem over cyclotomic rings ($Q=3329$) [cite: 548-555].
- [cite_start]**Key Encapsulation**: Simulates the KeyGen, Encapsulation, and shared secret derivation phases of the Kyber KEM [cite: 557-573].
- **PQC Focus**: Demonstrates the mathematical foundation of quantum-resistant security.

[cite_start]**Tech Stack:** Python, NumPy[cite: 545].
